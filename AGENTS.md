# Agent Instructions for DealerForge

## Goal
Build DealerForge CRM + AI Automation using TypeScript monorepo (Next.js / NestJS / Redis / Prisma / PostgreSQL / BullMQ).

## Rules
- Every API must enforce dealership_id tenancy.
- TypeScript and strong typing everywhere.
- Tests required for backend modules and core routes.
- Use clean modular design per domain module.
- Prioritize quality and correctness over speed.
- Commit diffs must be small and reviewable.
- No secrets checked into code (use env vars).
